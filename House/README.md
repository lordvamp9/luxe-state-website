# LuxeEstates

Welcome to **LuxeEstates**, a premium luxury real estate website designed to showcase exclusive properties with elegance and style.

## Project Structure

The project is organized as follows:

- **`index.html`**: The main landing page of the website.
- **`css/`**: Contains the stylesheet (`styles.css`) for the website's design.
- **`js/`**: Contains the JavaScript file (`script.js`) for interactivity.
- **`assets/images/`**: Stores all image assets used across the site.
- **`*.html`**: Individual property detail pages (e.g., `mansion-forest.html`, `penthouse-sky.html`, `villa-ocean.html`).

## How to Run

Since this is a static website, you can run it simply by opening the `index.html` file in any modern web browser.

1.  Navigate to the project folder.
2.  Double-click on `index.html`.

## Features

-   **Responsive Design**: Looks great on desktop and mobile devices.
-   **Interactive Elements**: Smooth scrolling and hover effects.
-   **Premium Aesthetics**: High-quality images and elegant typography.

## Tools Used

-   **HTML5**: Semantic structure.
-   **CSS3**: Custom styling, animations, and responsive layout.
-   **JavaScript**: Interactive functionality.
-   **Google Fonts**: Typography (Lato & Playfair Display).
